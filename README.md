📊 Healthcare Cost Analysis – 

Project Overview

This project focuses on analyzing healthcare expenses using data analytics techniques to identify patterns and factors affecting medical costs. The analysis is performed using Python, MySQL, and Power BI, combining data processing, querying, and visualization.

The dataset contains patient-related information such as demographic details, lifestyle factors, and medical charges. Through this project, meaningful insights are extracted to understand how different factors influence healthcare costs.

🎯 Objective

To analyze healthcare cost patterns
To identify key factors influencing medical expenses
To compare costs across different categories
To visualize insights using an interactive dashboard

🛠️ Tools & Technologies Used

Python – Data cleaning, analysis, visualization
MySQL – Data storage and query analysis
Power BI – Dashboard creation and data visualization

📌 Key Analysis Performed

Patient distribution analysis
Cost comparison based on lifestyle factors
Regional healthcare cost analysis
Relationship between BMI, age, and charges
Identification of high-cost patient groups

📊 Dashboard Features

KPI cards (Total Patients, Avg Charges, Max/Min Charges)
Cost comparison charts (Region, Smoking Status)
Distribution charts (Age, Gender, BMI, Children)
Scatter plots showing relationships between variables
Interactive filters for better data exploration

🔍 Insights

Lifestyle factors significantly impact healthcare costs
Certain regions show higher medical expenses
Higher BMI is associated with increased charges
Smoking leads to significantly higher healthcare costs

🚀 Conclusion

This project demonstrates how data analytics can be used to extract valuable insights from healthcare data. By combining Python, SQL, and visualization tools, the project provides a comprehensive understanding of cost patterns and helps in decision-making.
